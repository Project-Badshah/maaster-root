# Invite Maaster (`invite-maaster`)

## Overview
A beautiful, interactive, web-based digital invitation platform for modern events. It serves as an autonomous product that complements the `event-maaster` ecosystem.

## Architecture
- **Framework**: React + Vite
- **Routing**: `invite.maaster.online`
- **Identity**: Wired into `auth-maaster` via the `api.maaster.online` reverse proxy.
- **Styling**: Tailored event-specific palettes, high-fidelity micro-interactions.
