# Lyrics Maaster (RiLycs) (`lyrics-maaster`)

## Overview
RiLycs is an India-first unified live captioning, music recognition, and audio discovery platform. It transforms loudspeaker-powered events (from religious gatherings to concerts) into accessible, engaging experiences by displaying real-time captions and synced lyrics on audience devices.

## Architecture
- **Framework**: React Native (Expo)
- **Routing**: `lyrics.maaster.online` (Web access / PWA)
- **Database Architecture**: SQLite/Hive for offline usage (Pro Tier) and Neon Postgres for cloud syncing.

## The 4 Modes
1. **DVine**: Religious events (Mantras, slokas, prayers). "Every Prayer Visible"
2. **DCipher**: Music concerts (Bollywood, regional songs). "Every Lyric Live"
3. **DVerse**: Poetry events (Mushairas, spoken word). "Every Poem Preserved"
4. **DScribe**: Stage shows/events (Speeches, commentary). "Every Word Heard"
