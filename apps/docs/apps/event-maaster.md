# Event Maaster

## Overview
Event Maaster serves as the primary marketing gateway and storefront for the ecosystem.

## Architecture
Currently implemented as a **Next.js (App Router)** application.
- Prioritizes SEO optimization and server-side rendering.
- Designed with Apple-like aesthetics using TailwindCSS and Framer Motion.

## Future Path
While currently a Next.js storefront, Event Maaster will eventually transition into a SaaS vehicle powered by its own headless backend API, mirroring the HONC architecture of our other dashboards.
