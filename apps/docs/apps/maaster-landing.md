# Maaster Landing (`maaster-landing`)

## Overview
The public face and corporate directory of the Maaster Organization. It serves as the digital incubator's storefront, presenting the parent brand and showcasing the entire ecosystem of products.

## Architecture
- **Framework**: Next.js
- **Routing**: `maaster.online`
- **Role**: SEO-optimized landing page, routing users to individual SaaS products.
- **Styling**: Apple-like aesthetics, strict Tailwind, Framer Motion for micro-interactions.

## Domain Routing
This application sits at the root domain (`maaster.online`). All other autonomous applications sit on subdomains (e.g., `hisab.maaster.online`).
